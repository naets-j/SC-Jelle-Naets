🔧 Setup: Testimonials via Google Sheets

Volg deze stappen eenmalig. Daarna werkt alles automatisch.

Wat je nodig hebt

Een gratis Google-account (Gmail). Geen server, geen kosten, geen technische kennis vereist.

1Maak een Google Sheet aan

Ga naar sheets.google.com en maak een nieuw spreadsheet aan. Noem het bv. Testimonials Jelle Naets.

Zet in de eerste rij deze exacte kolomhoofdingen (kopieer en plak):

Timestamp
Naam
Titel
Bericht
Goedgekeurd

In kolom E (Goedgekeurd) staat TRUE voor zichtbare berichten. Nieuwe berichten van bezoekers komen automatisch als TRUE binnen — je kan dit later aanpassen naar FALSE om een bericht te verbergen.

Voeg nu je twee standaard-testimonials toe in rij 2 en 3:

automatisch
Een trotse supporter
Een trotse supporter
Eet ze daar allemaal…
TRUE
2Open de Apps Script editor

In je Sheet: klik bovenaan op Extensies → Apps Script. Er opent een nieuw tabblad.

Verwijder alle bestaande code en plak het volgende script volledig in: 

const SHEET_NAME = "Blad1"; // pas aan als jouw tabblad anders heet

// GET — leest goedgekeurde testimonials
function doGet(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet()
                              .getSheetByName(SHEET_NAME);
  const rows  = sheet.getDataRange().getValues();
  const data  = [];

  for (let i = 1; i < rows.length; i++) {
    const row         = rows[i];
    const goedgekeurd = String(row[4]).toUpperCase();
    if (goedgekeurd === "TRUE") {
      data.push({
        naam:    row[1],
        titel:   row[2],
        bericht: row[3]
      });
    }
  }

  return ContentService
    .createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}

// POST — schrijft nieuw bericht weg
function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet()
                              .getSheetByName(SHEET_NAME);
  try {
    const body = JSON.parse(e.postData.contents);
    if (body.action === "write") {
      sheet.appendRow([
        new Date(),   // Timestamp
        body.naam,    // Naam
        body.titel,   // Titel
        body.bericht, // Bericht
        "TRUE"        // Goedgekeurd
      ]);
    }
  } catch(err) {}

  return ContentService
    .createTextOutput(JSON.stringify({ ok: true }))
    .setMimeType(ContentService.MimeType.JSON);
}
📌 Nieuwe berichten van bezoekers komen automatisch in de sheet met Goedgekeurd = TRUE — ze zijn meteen zichtbaar op de website. Wil je een bericht verbergen of aanpassen? Dat doe je gewoon rechtstreeks in de Sheet.
3Publiceer het script als Web App

Klik rechtsboven op Implementeren → Nieuwe implementatie.

Stel in:

  • Type: Web-app
  • Uitvoeren als: Ik (jouw Google-account)
  • Toegang verlenen aan: Iedereen (dit is nodig zodat de website data kan lezen)

Klik op Implementeren. Google vraagt om toestemming — klik op Toegang verlenen en kies je account.

Je krijgt een Web App URL. Deze ziet er zo uit:

https://script.google.com/macros/s/AKfycb.../exec

Kopieer deze URL. Je hebt hem nodig in de volgende stap.

⚠\uefbf� Heb je het script al eerder gepubliceerd en nu aangepast? Kies dan Implementeren → Implementaties beheren en klik op het potloodicoon om een nieuwe versie te deployen. Een bestaande implementatie bijwerken zonder nieuwe versie pikt het gewijzigde script niet op.
4URL invullen in index.html

Open index.html in een teksteditor (bv. Notepad++ of VS Code). Zoek naar deze twee regels: 

const SHEET_READ_URL  = "JOUW_APPS_SCRIPT_URL_HIER";
const SHEET_WRITE_URL = "JOUW_APPS_SCRIPT_URL_HIER";

Vervang beide JOUW_APPS_SCRIPT_URL_HIER door jouw gekopieerde URL: 

const SHEET_READ_URL  = "https://script.google.com/macros/s/AKfycb.../exec";
const SHEET_WRITE_URL = "https://script.google.com/macros/s/AKfycb.../exec";

Sla het bestand op. Klaar!

5Berichten beheren

Wanneer een bezoeker een bericht instuurt via jouw website:

  • Het verschijnt automatisch in jouw Google Sheet én meteen op de website (Goedgekeurd = TRUE)
  • Wil je een bericht aanpassen? Klik gewoon op de cel in de kolom Naam, Titel of Bericht en typ je wijziging — na herladen van de website is het bijgewerkt
  • Wil je een bericht verbergen? Zet Goedgekeurd op FALSE
  • Wil je een bericht volledig verwijderen? Verwijder de hele rij
💡 De volgorde in de sheet bepaalt de volgorde op de website. Sleep rijen om de volgorde te wijzigen. De laatste 4 goedgekeurde berichten verschijnen als grote kaarten, de rest in de scrollbare lijst.
Samenvatting: bezoekers sturen een bericht in → verschijnt meteen op de website én in jouw Sheet. Vanuit de Sheet kan je teksten aanpassen, verbergen of verwijderen wanneer je maar wil. Geen code nodig.
